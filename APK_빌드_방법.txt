★ 앱 기능 변경 (URL 설정이 필요 없어졌습니다!)
────────────────────────────────────────────
앱을 처음 실행하면 서버 URL 입력 화면이 나타납니다.
Railway 배포 후 생성된 URL을 입력하면 바로 연결됩니다.
(한 번 입력하면 저장되어 다음부터는 바로 실행)


[ 방법 1 ] GitHub Actions 자동 빌드 ★ 추천
═══════════════════════════════════════════

준비물: GitHub 계정 (https://github.com 무료 가입)

STEP 1. GitHub에서 새 저장소 만들기
  1. https://github.com/new 접속
  2. Repository name: sns-expert (또는 원하는 이름)
  3. Public 또는 Private 선택
  4. "Create repository" 클릭
  5. 생성된 저장소 URL 복사
     예시: https://github.com/내아이디/sns-expert.git

STEP 2. github_올리기.bat 실행
  - 이 폴더에 있는 github_올리기.bat 더블클릭
  - GitHub 저장소 URL 붙여넣기
  - 엔터 → 자동으로 코드 업로드

  (Git 없으면 https://git-scm.com/download/win 에서 설치)

STEP 3. APK 다운로드
  업로드 후 5~10분 기다리면 자동 빌드 시작!

  1. https://github.com/내아이디/sns-expert 접속
  2. 상단 "Actions" 탭 클릭
  3. "APK 빌드" 클릭
  4. 완료된 실행(초록 체크 ✅) 클릭
  5. 페이지 하단 "Artifacts" → "SNS전문가-앱-debug" 클릭
  6. ZIP 다운로드 → 압축 풀면 app-debug.apk 파일


[ 방법 2 ] Android Studio 로컬 빌드
═══════════════════════════════════════════

1. Android Studio 설치
   → https://developer.android.com/studio

2. android_app 폴더를 Android Studio로 열기

3. Build → Build APK(s) 클릭

4. APK 위치:
   android_app/app/build/outputs/apk/debug/app-debug.apk


[ APK 폰에 설치하기 ]
═══════════════════════════════════════════

1. 폰 설정 → 보안 → "출처를 알 수 없는 앱 설치" 허용
2. APK 파일을 카카오톡/USB로 폰에 전송
3. 파일 관리자에서 APK 탭하여 설치
4. 앱 실행 → 서버 URL 입력 (Railway 배포 URL)


[ 서버 배포가 안 되어 있다면 ]
═══════════════════════════════════════════

APK는 웹 서버에 접속하는 앱입니다.
서버 배포 없이는 앱이 동작하지 않습니다.

서버 배포 방법:
  - android_app/APK_빌드_가이드.txt 의 [STEP 1] 참고
  - Railway.app 에 무료로 배포 가능 (월 $5 크레딧 제공)
